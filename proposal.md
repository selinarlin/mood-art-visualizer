# Mood-Based Generative Art Visualizer

## Repository
<https://github.com/selinarlin/mood-art-visualizer>

## Description
This interactive project will generate animated swirls and colors based on the user's selected mood. 
It will demonstrate how programming can create expressive visuals through motion, color, and patterns.

## Features
- Mood Selection
	- Users will be able to switch between their desired mood, such as calm, chaotic, nostalgic, sad, etc., by pressing on each mood's associated number key.
- Swirl Animation
	- This project will utilize the particle system and manipulate it to follow swirl/curved paths.
- Dynamic Colors
	- Each mood will follow a different mood-tied color scheme to create its own unique visual style.
- Motion Variations
  - particle/swirl speed will vary depending on the mood.

## Challenges
- Learning how to create the swirl/curved motion using math.
- Learning how to add/link keyboard inputs to switch between moods in real time.
- Expanding particle system into a swirl motion system.

## Outcomes
Ideal Outcome:
- A smooth-running, interactive art visualizer with multiple moods, each creating a visually appealing animation.

Minimal Viable Outcome:
- A working animation with at least two mood options that change particle colors and movements.
